Excel vs. Python: E-Commerce Sales Analysis

A comparative analysis of a 2023 e-commerce dataset using Excel Pivot Tables and Python (Pandas + Matplotlib). The project evaluates how different analytical tools affect workflow, scalability, and reproducibility while producing the same business insights.

Project Overview

This project evaluates Excel Pivot Tables and Python (Pandas + Matplotlib) by answering identical business questions on a 2023 e-commerce dataset. The goal is to compare analytical workflows, scalability, reproducibility, and insight generation in a realistic business context.

Business Findings

  • Revenue concentration: Revenue is concentrated in Electronics, Home & Garden, and Sports.
  • Discounts: There is no strong evidence that discounts independently increase order value.
  • Retention: Returning customers drive revenue primarily via higher purchase frequency.
  • Device: Mobile dominates order volume; desktop has slightly higher average order value.
  • Delivery: Delivery speed shows minimal relationship with customer ratings.

Both tools produced consistent business conclusions. The primary differences emerged in workflow efficiency, automation, and long-term scalability.

Excel vs. Python: Workflow Comparison

  • Speed for ad-hoc analysis: Excel Pivot Tables allowed rapid aggregation and exploration of the dataset with minimal setup.
  • Automation and reproducibility: Python (Pandas) required more initial setup but enabled a fully reproducible workflow where data cleaning, aggregation, and visualization can be reproduced or automated.
  • Scalability: Excel performed well for this moderately sized dataset but is likely to become cumbersome with larger datasets. Python is a better option for larger datasets and repeatable pipelines at scale.
  • Visualization: Excel provides fast built-in charting, while Python (Matplotlib) offers greater control.
  • Practical takeaway: Excel is ideal for quick exploratory analysis and stakeholder reporting, while Python is better suited for repeatable analytical workflows and larger data environments.
