SURF Projects

Undergraduate research exploring procedural content generation using diffusion models and large language models (LLMs).

Mario Diffusion

Trained diffusion models with UNet architecture for Procedural Content Generation (PCG) for game level design with optional text guidance. Contributed to model training infrastructure (early stopping, checkpoint resumption), enhanced tools for mixed-initiative level generation, and developed scripts to evaluate and visualize model runtime and level solvability using A* metrics.
